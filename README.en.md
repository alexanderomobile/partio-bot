# PartioBot

**Telegram bot for shared expense tracking**

> Showcase only (no source code). Code in a private repository.

---

## Problem

Telegram bot for shared expense tracking.

## Features

Events, invite codes, expenses, balances, transfer minimization, group chats.

## Business value

Saves time settling group bills after trips and events; replaces messy spreadsheets in chats.

## Stack

Python · aiogram 3 · FastAPI · SQLAlchemy · MySQL

## Architecture

![Architecture](assets/partio-bot-diagram.svg)

## Run

```bash
pip install -e ".[dev]" && partiobot
```
