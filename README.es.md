# PartioBot

**Bot de Telegram para gastos compartidos**

> Showcase sin código. Fuentes en repositorio privado.

---

## Problema

Bot de Telegram para gastos compartidos.

## Funcionalidad

Eventos, códigos, gastos, balances, minimización de transferencias, chats grupales.

## Valor de negocio

Ahorra tiempo al cerrar cuentas de grupo; elimina hojas de cálculo en el chat.

## Stack

Python · aiogram 3 · FastAPI · SQLAlchemy · MySQL

## Arquitectura

![Architecture](assets/partio-bot-diagram.svg)

## Ejecución

```bash
pip install -e ".[dev]" && partiobot
```
